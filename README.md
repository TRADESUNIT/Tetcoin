#### **Tetcoin (TET)**

Open source peer to peer currency.
* Scrypt Algorithm
* 4 minute block targets
* subsidy halves in 544k blocks (~4 years)
* ~128 million total coins
* 128 coins per block
* 2016 blocks to retarget difficulty
